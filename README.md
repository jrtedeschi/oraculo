# Oráculo 

Uma aplicação em CLI e WebApp para transcrição de Áudios e *Busca Semântica*


## Requisitos
 - python 3.10
 - ffmpeg
 - git
 - docker (futuro)

## Instalação
```bash
pip install oraculo
```


## Uso 
### Iniciar a Aplicação de Busca
```bash
oraculo webapp
```

### Iniciar Transcrição
De um único arquivo
```bash
oraculo transcribe
```

De uma pasta
```bash 
oraculo bulk-transcribe
```
